Empty State Placeholder Images
================================

This directory contains placeholder images for empty states in the Plan Detail modal.

Files:
------
1. master-plan-set.svg
   - Used in: Plan Detail > Overview > Master Plan Set section
   - Shows when no master plan set file is uploaded
   - Replace with actual PDF/file icons illustration

2. no-communities.svg
   - Used in: Plan Detail > Overview > Communities table
   - Shows when no communities are using this plan set
   - Replace with actual community/building icon illustration

3. no-projects.svg
   - Used in: Plan Detail > Overview > Projects table
   - Shows when no projects are using this plan set
   - Replace with actual folder/project icon illustration

Instructions:
-------------
To replace these placeholders with your actual images:
1. Design or obtain the actual empty state illustrations
2. Save them with the same filename (or update the references in PlanDetailModal.tsx)
3. Recommended format: SVG (scalable) or PNG (64x64px or larger)
4. Keep the files in this same directory: /public/assets/empty-states/
