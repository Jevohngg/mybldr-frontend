Material Images Directory Structure
=====================================

This directory contains placeholder image paths for the Custom Palette Editor.
Replace these placeholder paths with your actual material images.

----------------

Main Siding:
- main-siding/aluminum.jpg
- main-siding/stucco.jpg
- main-siding/wood-grain.jpg
- main-siding/wood-shake.jpg

- accent-siding/brick.jpg
- accent-siding/stone.jpg

Roofing:

- roofing/shingle-light.jpg

Masonry:

- masonry/stone-gray.jpg

Sink:
- sink/stainless.jpg
- sink/white.jpg

Preview Mockup:
- palette-preview-mockup.jpg (appears in the upload section)

Recommended Image Specifications:
----------------------------------
- Format: JPG or PNG
- Dimensions: 400x300px minimum
- Aspect Ratio: 4:3 or similar
- File size: Keep under 200KB for optimal loading

To update images:
-----------------
1. Create the subdirectories listed above inside public/assets/materials/
2. Add your material images with the exact filenames shown
3. The images will automatically appear in the Custom Palette Editor
