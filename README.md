# myBLDR Community Prototype (Frontend-only)

High-fidelity, fully clickable prototype (no backend). Built with Vite + React + TypeScript.

## Run locally
1. Open this folder in Cursor
2. Install deps:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

## What’s implemented
- Top nav + left nav (matches mockups)
- Communities page with “+ Add Community” modal
- Community detail shell + side nav tabs
- Community overview with stat cards, plan cards, map section
- “Add plans to community” modal (select + Done)

## Notes
- All data is mocked in `src/mock-data/`
